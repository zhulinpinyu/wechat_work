# WechatWork

## Installation

```elixir
def deps do
  [
    {:wechat_work, "~> 0.2"}
  ]
end
```

## usage

```elixir
WechatWork.push("@all", "测试一下")
```

Notes: **please setup env params** `./envrc.sample`
